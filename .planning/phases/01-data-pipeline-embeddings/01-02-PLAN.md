  <action>
**Part A: Model architecture** (`backend/ml/embedding_model.py`)

Create `PlayerEmbeddingModel(nn.Module)` following the entity embedding MLP pattern from research:

1. **Embedding layers:**
   - `player_embed = nn.Embedding(n_players, embed_dim)` where embed_dim=8 (start high within 4-8 range, reduce if overfitting)
   - `map_embed = nn.Embedding(n_maps, 4)` -- 12 maps, 4 dims captures map characteristics
   - `role_embed = nn.Embedding(n_roles, 3)` -- 4 roles, 3 dims sufficient

2. **MLP head:**
   - Input: embed_dim + 4 + 3 + n_continuous (8+4+3+8 = 23 total)
   - Hidden layers: Linear(23, 64) -> ReLU -> Dropout(0.2) -> Linear(64, 32) -> ReLU -> Dropout(0.2) -> Linear(32, 1)
   - Output: single scalar (predicted kills)

3. **Methods:**
   - `forward(self, player_idx, map_idx, role_idx, continuous)` -- standard forward pass
   - `get_player_embedding(self, player_idx)` -- extract and detach player embedding vector
   - `get_all_embeddings(self)` -- return the full player embedding weight matrix (detached)

**Part B: Training loop** (`backend/ml/train.py`)

1. **collate_fn(batch)**: Custom collate that stacks the dict-based dataset items into batched tensors.

2. **train_model(db_path, embed_dim=8, lr=0.001, epochs=50, batch_size=256, patience=10, model_dir='models')**:
   - Call `create_datasets(db_path)` to get train/test datasets + metadata
   - Instantiate `PlayerEmbeddingModel` with metadata dimensions
   - Use `torch.optim.Adam(model.parameters(), lr=lr)`
   - Use `nn.MSELoss()` for kill prediction
   - Training loop with DataLoader (batch_size, shuffle=True for train)
   - Track train_loss and val_loss per epoch
   - Early stopping: stop if val_loss hasn't improved for `patience` epochs
   - Log progress every 5 epochs: epoch, train_loss, val_loss
   - After training, save: model state_dict to `{model_dir}/player_embeddings.pt`, metadata (mappings, norm stats, embed_dim, loss history, split info) to `{model_dir}/training_meta.json`
   - Return dict with: model, metadata, history (train_loss list, val_loss list), best_epoch

3. **evaluate_model(model, dataset, batch_size=256)**:
   - Compute MSE and MAE on the given dataset
   - Return dict with mse, mae, rmse

4. **load_model(model_dir='models')**:
   - Load training_meta.json for dimensions
   - Instantiate model and load state_dict
   - Return (model, metadata)

2. Create `scripts/train_embeddings.py` -- CLI entry point for training:
   ```python
   """Train player embedding model from SQLite data."""
   import sys
   import os
   sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))

   import argparse
   from backend.ml.train import train_model, evaluate_model
   from config import Config

   def main():
       parser = argparse.ArgumentParser(description='Train player embedding model')
       parser.add_argument('--db-path', default=Config.DATABASE_PATH, help='Path to SQLite database')
       parser.add_argument('--embed-dim', type=int, default=8, help='Player embedding dimension (4-8)')
       parser.add_argument('--epochs', type=int, default=50, help='Maximum training epochs')
       parser.add_argument('--batch-size', type=int, default=256, help='Training batch size')
       parser.add_argument('--lr', type=float, default=0.001, help='Learning rate')
       parser.add_argument('--patience', type=int, default=10, help='Early stopping patience')
       parser.add_argument('--model-dir', default='models', help='Directory to save trained model')
       args = parser.parse_args()

       result = train_model(
           db_path=args.db_path,
           embed_dim=args.embed_dim,
           lr=args.lr,
           epochs=args.epochs,
           batch_size=args.batch_size,
           patience=args.patience,
           model_dir=args.model_dir,
       )

       # Print summary
       history = result['history']
       print(f"\nTraining complete:")
       print(f"  Best epoch: {result['best_epoch']}")
       print(f"  Final train loss: {history['train_loss'][-1]:.4f}")
       print(f"  Final val loss: {history['val_loss'][-1]:.4f}")
       print(f"  Best val loss: {min(history['val_loss']):.4f}")

       # Evaluate
       eval_result = evaluate_model(result['model'], result['test_dataset'])
       print(f"\nTest set evaluation:")
       print(f"  RMSE: {eval_result['rmse']:.2f} kills")
       print(f"  MAE: {eval_result['mae']:.2f} kills")

       # Quick embedding sanity check
       model = result['model']
       meta = result['metadata']
       print(f"\nEmbedding space:")
       print(f"  Players: {meta['n_players']}")
       print(f"  Embedding dim: {args.embed_dim}")

       # Show a few example embeddings
       import torch
       sample_players = list(meta['player_to_idx'].items())[:5]
       for name, idx in sample_players:
           vec = model.get_player_embedding(torch.tensor([idx]))
           print(f"  {name}: {vec.numpy().flatten()[:4]}...")

   if __name__ == '__main__':
       main()
   ```
